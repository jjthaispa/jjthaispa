# J.J Thai Spa Website

This repository contains the source code for the J.J Thai Spa website, built with React, Vite, and Tailwind CSS, and hosted on Firebase.

## Quick Start (Local)

1. **Install dependencies**: `npm install`
2. **Run the app**: `npm run dev`

For detailed instructions on environment setup and deployment, please see [DEPLOYMENT.md](./DEPLOYMENT.md).
